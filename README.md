Game simulasi akuarium berbasis Unity di mana pemain mengelola ekosistem ikan secara real time. Game ini menggunakan AI untuk penulisan code dari ChatGPT untuk sistem spawn berbasis file dan konfigurasi gameplay yang bisa diubah tanpa harus rebuild. 

Cara membuka game : 
Jalankan Aquascape.exe 

Anda dapat mengubah asset sesuai yang anda inginkan dengan cara : 
Lakukan edit pada file config.json yang berlokasi di AquascapeData/StreamingAssets/config.json 

Format asset yang hanya bisa diterima ialah .png, dan format penamaan filenya hanya FISH[TYPE][YYYYMMDDHHMMSS].png untuk ikan dan TRASH[TYPE]_[YYYYMMDDHHMMSS].png untuk sampah 

Contoh penamaan file ikan : FISH_MACKEREL_20260401165920.png 
Contoh penamaan file sampah : TRASH_BOTTLE_20260401165920.png
